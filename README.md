## Persona
O aplicativo foi projetado para **estudantes que moram em repúblicas** ou dividem apartamento com amigos.

Esses usuários frequentemente precisam dividir contas como:
- Luz
- Internet
- Aluguel
- Compras de mercado

Eles enfrentam problemas como:
- Esquecer pagamentos
- Erros na divisão
- Discussões sobre valores

---

## Heurística de Nielsen em foco
**Visibilidade do status do sistema**

O protótipo prioriza mostrar claramente:
- Quem deve
- Quanto deve
- Para quem deve

Isso reduz a ansiedade do usuário e evita confusão.

---

## Fluxo de Tarefa
### Cadastro de uma conta de luz e divisão entre 3 pessoas:

1. O usuário acessa o grupo "Casa"
2. Clica no botão **"Adicionar gasto"**
3. Insere:
   - Descrição: Conta de luz
   - Valor: R$150
4. Seleciona a opção **dividir igual**
5. Marca as 3 pessoas envolvidas
6. Clica em **"Salvar gasto"**
7. O sistema calcula automaticamente:
   - R$50 para cada pessoa
8. O valor aparece no extrato de dívidas

---

## Objetivo do Aplicativo
Facilitar a divisão de contas entre pessoas, trazendo:
- Clareza
- Organização
- Menos conflitos
